Overview

I want to try to keep the .eslintrc.json file found in my Github gist as the primary source: http://bit.ly/eslint-elf

To insert it into your current directory, you should be able to run the following script stored in JsObjects:

get-eslintrc

A symbolic link in your ~/bin directory should point to this script. This means you should be able to simply type get-eslintrc from any directory on your system and then press enter to run the script. (You'll need write permissions in the current directory.)

Creating a Link to the Script

If a symbolic link to get-eslintrc is not in your ~/bin directory, then do the following:

Command Explanation
slb Go to ~/Git/JsObjects/Utilities/SetupLinuxBox
git pull get the latest JsObjects
./CreateSymbolicLinks create the ~/bin/get-eslintrc symbolic link.

The original script is found here:

~/Git/JsObjects/Utilities/Templates/React/get-eslintrc
