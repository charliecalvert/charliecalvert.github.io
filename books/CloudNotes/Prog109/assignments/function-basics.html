Function Basics

Simple Functions

Inside the javascripts-tests directory create function-basics.html and function-basics.html. Add a script statement to the head element of the HTML file that links in the JavaScript file.

Create a link from your home page to function-basics.html.

In what follows, When I ask you to print something out, use console.log to print it.

  • Create a function called simpleFunction that prints out the phrase: "functions should consist of statements designed to perform a single task."
  • Create a function called functionParameters that takes a single parameter and prints it out.
  • Create a function called functionReturn that returns the string "Many functions return values."

At the bottom of function-basics.js call all three functions. Pass the following string to functionParameters: "Many functions take parameters."

Show Results in Html

In function-basics.html create an H1 element with the content Function Basics.

Add three empty paragraph elements. Give them the following IDs:

  • simple-function
  • function-parameters
  • function-return

Add a button with the content Call Functions

In function-basics.js respond to clicks on the button by calling all three functions.

Modify the functions so that, in addition to the console.log statements, they also set the innerHtml of each paragraph in function-basics.html
