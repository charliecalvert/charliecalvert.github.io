Code to sort our Array of Object by rank:



// Code omitted here        
arrays.languagePopularity.sort(compareRank);
// Code omitted here

function compareRank(a, b) {
     const rankA = a.rank;
    const rankB = b.rank;

    let comparison = 0;
    if (rankA > rankB) {
        comparison = 1;
    } else if (rankA < rankB) {
        comparison = -1;
    }
    return comparison;
}

When we ask to sort our array of objects, we need to tell the JavaScript sort method how to order the items in the array. The compareRank function is our way of telling JavaScript how to order the items when it sorts them.

Adding List Items to an Unordered List

The code we care about most in the excerpt found in the previous section is in the for...of loop. In that code we:

  • Use document.createElement to create a new list item. We are using JavaScript to do the same thing we do in HTML when we write <li>. We are dynamically creating an HTML element in JavaScript code. After the line executes, we have an empty list item:
    • <li></li>
  • The next line uses document.createTextNode to create a text object that can be appended onto a list item. This is similar to what we do when we write foo.textContent = '...' in our Object Choices propgram. But this time, instead of assigning a value to the textContent property of a paragraph, we are creating a textNode that contains the text we want to display, and inserting (appending) it to the list item. Now we have a list item with some text in it:
    • <li>My Text</li>
  • Finally, in the last line, we write numbersDisplay.appendChild(li) and append the list item to the list on our HTML page.
