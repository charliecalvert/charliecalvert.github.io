Link Exercises

Create Useful Links

Create a page called useful-links.html in the root of your repository. Link to it from your home page.

In useful links, create links to the following:

  • Google
  • Stack Overflow
  • MDN
  • W3Schools
  • Lynda

Clean up Home Page

Make sure your home page links all work. Divide it up into our three or four key sections:

  • Header and Nav
  • Main
  • Footer

Turn it in

Provide a link to your home page on GitHub Pages, which should in turn link to the useful links page. Remember, this is not the same thing as your repository. This is your website.

Also provide a link to your repository.
