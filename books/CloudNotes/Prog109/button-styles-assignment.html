Style Button Elements

  • In the ListButton directory create a style sheet called styles.css.
  • Place code for styling buttons in it.
    • The deck in the References section provides a sample CSS rule for the button element. Model your work on it, but change the values of each property to suit your tastes.
    • Don't use the same values as I do.
    • Don't add any other declarations or properties at this point.
    • In other words, your rule should have exactly five declarations in it.
    • The declarations should be the same as those in the Button Styles deck.
  • Link your style sheet into ListButton/index.html
