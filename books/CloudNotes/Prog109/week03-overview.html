Overview

We will start with review and with picking up a view subjects that we have skipped over. We will also briefly discussing some of the new HTML5 elements and review images and how to validate our HTML documents. This will lead us right into the basics of Cascading Style Sheets. This module contains a few pages on the CSS Box Model; using colors with HTML, and formatting text on a page. 

  • File naming conventions. All small, no spaces, dashes rather than underscores.
  • Proper nesting of tags
    • Don't put h1 or ul inside a p tag.
  • Lists (ul and ol tags) with list items.
  • Inline vs Block
  • id and class
  • HTML Validator
  • Images, Image Elements and Copyright.
  • Center and CSS
  • The address tag.
  • .gitignore
  • No code in the discussion area.

    • Module Objectives

    • Create a web page using a basic HTML5 page structure with HTML5 elements
    • Learn more about external and internal  CSS stylesheets 
    • Use validation services to check the syntax of HTML pages
    • Describe the most common CSS selectors
    • Learn about id and class attributes
    • Demonstrate using colors for text and background on a web page.
