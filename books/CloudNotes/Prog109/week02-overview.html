Overview

Welcome to Programming 109 week 2! This week we will focus on:

  • The structure of an HTML document
  • The most important HTML elements
  • CSS

In addition, we will continue to learn about new HTML 5 elements.

Additional Possible Module Objectives

  • Demonstrate an understanding of directory and file structure
  • Describe the three image formats used on the web
  • Describe how to properly display an image, and link with it
  • Adopt proper formatting techniques (like indentation and spacing) when coding HTML documents
