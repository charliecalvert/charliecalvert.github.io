Body

Put your content in the <body> tag. There is only one <body> tag in each HTML document. It is the second element in an <html> element.

                <html>
                    <body>
                    </body>
                </html>

Nav

Put your navigation links in a <nav> element. Usually a:

  • Table of Content (TOC),
  • index, menu

foo 

					
<nav>
	<ul>
		<li>qux</li>
		<li>baz</li>
		<li>xyzzy</li>
	</ul>
</nav>
