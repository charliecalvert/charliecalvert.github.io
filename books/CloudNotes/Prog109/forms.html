Overview

In this module we will cover the nuts and bolts of forms and demonstrate various form fields. HTML5 introduced some snazzy new form features that we'll take a look at. The downside is they are not uniformly supported on all browsers, but the situation is improving day by day.  

Module Objectives

  • Demonstrate how to code an HTML form
  • Describe the difference between POST and GET 
  • Utilize form variables and values 
  • Develop a page layout with float and flex
  • Continue developing content using CSS
