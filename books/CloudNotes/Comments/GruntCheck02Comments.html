Comments on Grunt Check02

Sanity Check

Run git pull to ensure you have the latest from JsObjects. Copy grunt-check-two into ~/bin:

cp%20%24JSOBJECTS/Utilities/NodeInstall/grunt-check-two%20%7E/bin/.

Run the script and make sure it comes back clean, with all text in green.

Npm Test

Be sure you include support for npm test in package.json:

 "scripts": {
    "start": "nodemon ./bin/www",
    "test": "karma start"
  },

Error Handler

Some of you forgot to change the the error handler in app.js so you can see what code is throwing development errors.
