Tag

Create a tag marking your current status:

$ git tag -a v3.0.0 -m "Start Week03"
$ git push origin v3.0.0
$ git tag -n1

The last command lists your tags and their message on one line

Bower

Change bower.rc to reference public/bower-components

Add bower-components to .gitignore:

# Dependency directories
node_modules
jspm_packages
bower-components

New Branch

Create a new branch called Week03. You must put your homework for week three in this branch! (You can do it!)

git branch Week03
git checkout Week03

Write some code, commit your work. Push it:

git push --set-upstream origin Week03
