Description

Create a MangoExpress project on a subject of your choosing. This is simply the next step on the project we have been developing.

NOTE: Though I wasn't intentionally following any plan, but just doing what seemed obvious, nevertheless the type of project we have been developing seems to be called a Mean Stack project: MongoDb + Express + Angular = MEAn. MangoExpress is just another way to combine portions of the words MongoDb, Express, and Angular into a single word.

Step One

Copy your AngularFacadeUi project into a new folder called Week07-MangoExpress. A command to accomplish this goal might look something like this:

robocopy Week06-TestLive02 Week07-MangoExpress /MIR

Open your "new" project in WebStorm.

Step Two

Add in bootstrap.

bower install jquery --save
bower install bootstrap --save

Add bootstrap to layout.jade:

Turn your buttons into Bootstrap buttons. We start with this:

button(ng-click='myController.loadScientists()') Load Scientists

Add the btn and btn-default classes to your button:

button(class="btn btn-default" ng-click='myController.loadScientists()') Load Scientists

Turn it in

Make sure your code is in the Week07-MangoExpress folder of your repository. Confirm that all your tests pass. Press the submit button and add comments as usual or as needed.
