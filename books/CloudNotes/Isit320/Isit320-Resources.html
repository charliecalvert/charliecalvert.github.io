Interesting Books

Here is a beginning list of books available on the 24X7 site and elsewhere that you might find useful. If the book is not on 24X7, then I would not put it here unless I thought it was a great book. The books on the 24X7 site are all free, and so I include most of the ones connected to our course, regardless of quality. If I see one from 24X7 that I think is particularly good. I will put an asterisk after it.

Please check back from time to time, as I intend to keep adding to this list.

###JavaScript

  • JumpStart JavaScript 24X7
  • JumpStart NodeJs 24X7

###Web and Networks

  • Web Application Hacker's Handbook* (24X7)
  • Networking All-in-One for Dummies, 5th Edition (24X7)
  • Ubuntu Linux Toolbox (24X7)
  • Linux Bible, 8th Edition (24X7)

###Books not on 24X7

  • JavaScript the Good Parts (Crockford, Oreilly) - This is classic, but a bit hard to read.
  • JavaScript Ninja (Resig, Manning) - Not as good an introduction as the Good Parts, but easier to understand and more depth.
  • Maintainable JavaScript (Zakas, O'Reilly) - The most approachable of the three.
