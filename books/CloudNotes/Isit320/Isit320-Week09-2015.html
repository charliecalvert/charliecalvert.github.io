Working with Bitmaps

Animate changes to a sprite when it's state changes. In Crafty 06 you can find examples of how to do this. The key code is in Food.js, particularly the calls this.sprite:

visit: function() { 'use strict';
    this.count++;
    switch (this.count) {
        case 1:
            this.sprite(1, 0);
            break;
        case 2:
            this.sprite(2, 0);
            break;

        case 3:
            this.sprite(3, 0);
            break;

        default:
            this.destroy();
            break;
    }

See also scenes.js:

Crafty.sprite(32, assets[0], {
    spr_tree:    [0, 3],
    spr_bush:    [1, 3],
    spr_village: [0, 1],
    spr_food: [0, 0]            
});
